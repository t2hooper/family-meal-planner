📱 Install on Your Phone or Tablet

✅ You can install right now!

Your browser supports direct install. Click the button below:

🍎 iPhone / iPad (Safari)
  1. Open this file in Safari (not Chrome)
  2. Tap the Share button ⎦↑⎡ at the bottom of the screen
  3. Scroll down and tap "Add to Home Screen"
  4. Tap Add — done! Look for the 🥗 icon on your home screen
🤖 Android (Chrome)
  1. Open this file in Chrome
  2. Tap the three-dot menu ⋮ in the top-right corner
  3. Tap "Add to Home Screen" or "Install App"
  4. Tap Install — the 🥗 icon will appear on your home screen
💻 Desktop (Chrome / Edge)
  1. Open this file in Chrome or Edge
  2. Look for the install icon ⊕ in the address bar (right side)
  3. Click it and confirm — the app opens in its own window

💡 Tip: Both adults should install from the same shared Google Drive link so you're always opening the latest synced version.
