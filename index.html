Costco Items — tagged on the shopping list as a separate Costco trip

Items you buy at Costco. When a recipe's ingredient matches one of these, it's pulled into the Costco list (and Costco export) instead of the regular grocery store trips. Matching is by whole word, so "chicken breast" here will catch "boneless chicken breast" in a recipe.

Recipe Names — tidy existing names so the main item sorts first

New recipes are auto-named with the primary meal item first (e.g. Maple Salmon → Salmon, Maple) and filler words removed. Run this to apply the same tidy-up to recipes already in your library. You'll see every proposed change and can deselect any before saving.

Data Management

📱 Install on Your Phone or Tablet

Add this app to your home screen so it opens like a real app — no browser bars, full screen, instant access.

✅ You can install right now!

Your browser supports direct install. Click the button below:

🍎 iPhone / iPad (Safari)
  1. Open this file in Safari (not Chrome)
  2. Tap the Share button ⎦↑⎡ at the bottom of the screen
  3. Scroll down and tap "Add to Home Screen"
  4. Tap Add — done! Look for the 🥗 icon on your home screen
🤖 Android (Chrome)
  1. Open this file in Chrome
  2. Tap the three-dot menu ⋮ in the top-right corner
  3. Tap "Add to Home Screen" or "Install App"
  4. Tap Install — the 🥗 icon will appear on your home screen
💻 Desktop (Chrome / Edge)
  1. Open this file in Chrome or Edge
  2. Look for the install icon ⊕ in the address bar (right side)
  3. Click it and confirm — the app opens in its own window

💡 Tip: Both adults should install from the same shared Google Drive link so you're always opening the latest synced version.
